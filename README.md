## Descripción
Este proyecto se realiza como parte de un trabajo práctico de ofimática, en el que se investigan y utilizan distintas herramientas digitales para la organización, creación y presentación de información.

El objetivo es entender mejor cómo funcionan estas herramientas y aprender a trabajar en equipo de forma organizada, utilizando la metodología Scrum.

Como resultado final, se elaborará un informe y/o una presentación con la información recopilada durante el desarrollo del trabajo.
