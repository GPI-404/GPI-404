## Descripción
Este proyecto consiste en el desarrollo de un trabajo práctico de ofimática, donde se investigan y aplican herramientas digitales para la organización, creación y presentación de información.

El objetivo es comprender el uso de estas herramientas y trabajar de manera colaborativa utilizando la metodología Scrum.

Como resultado, se realizará un informe y/o presentación con la información recopilada.

## Metodología
Se utilizó la metodología Scrum:
- Se definieron tareas (Product Backlog)
- Se organizaron en un Sprint
- Se realizó seguimiento del progreso
